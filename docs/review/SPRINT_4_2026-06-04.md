## Bài học rút ra (cho Sprint 5+)

1. **Per-field merge thắng all-or-nothing.** Mỗi phase (4.E/F/G) chọn per-field merge → AI có thể emit partial, fallback legacy per field. Friendly hơn rất nhiều.
2. **`bool` field default `False` là blocker.** SubtitlePolicy.emphasis_pass và CameraStrategy.motion_aware_crop đều phải defer cùng lý do. Future schema cần explicit sentinel hoặc `Optional[bool]`.
3. **Env-flag gate trong resolver thắng module-level check khi flag rò rỉ.** F.5 (Sprint 4.G) cho thấy shim leak có thể vi phạm Contract #2 dù flag check ở orchestrator level. Resolver tự gate là deterministic + Contract-safe.
4. **Planner subagent giá trị cao cho CRITICAL phases.** Mỗi phase Planner phát hiện ít nhất 1 fact sai trong brief Leader: ranking decision ở render_pipeline.py không phải pipeline_ranking.py (4.G); 2 CameraStrategy classes không clash runtime (4.F); shim ranks luôn = enumerate (F.5 4.G).
5. **Source-level pin tests rẻ + đáng tin.** Mỗi consume phase có 4-5 source-level grep tests pin event names + source ordering + import existence. Catch refactor regressions cheap.

## Migration completion

Sprint 4 cuối cùng pin AI Director Architecture trong codebase. Mô hình hiện tại:

```
Local Video → Transcript → CreatorContext Builder → AI Director (full RenderPlan emit)
                                                              ↓
                                          RenderPlan (persisted as render_plan_json)
                                                              ↓
                                                  PartRenderContext.render_plan
                                                              ↓
                                          Render Engine (pure executor consuming):
                                            • subtitle_policy.style + market (4.E)
                                            • camera_strategy.reframe_mode (4.F)
                                            • ClipPlan.rank (4.G, env-flag gated)
```

Khi `LLM_EMIT_RENDER_PLAN=1`: AI Director quyết, Render Engine thực thi.
Khi `LLM_EMIT_RENDER_PLAN=0` (default): Render Engine chạy legacy logic byte-identical baseline.
