Channel folder structure
========================

video_out/
  - Final videos for upload (primary source for Upload mode)

upload/source/
  - Downloaded original source videos
upload/queue/
  - Optional queue staging
upload/uploaded/
  - Uploaded successful files
upload/failed/
  - Upload failed files
upload/archive/
  - Archived files
upload/logs/
  - Upload runtime logs

account/upload_settings.json
  - Channel-level upload defaults (proxy/network/browser)
account/profiles/{account_key}/account.json
  - Per-account config/profile
account/profiles/{account_key}/browser-profile/
  - Persistent browser user data for that account
  - First-time profile folder is auto-named by account_key:
    chromeportable_{account_key} or firefoxportable_{account_key}
account/templates/credential_line_template.txt
  - One-line credential format reference

browser-profile/
  - Place portable browser binaries here (FirefoxPortable/ChromePortable)

logs/upload/
logs/render/
